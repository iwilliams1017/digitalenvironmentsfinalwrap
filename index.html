Key Priorities in IT 248

1. System Design, Engineering and Organization

To create effective digital environments, developers should integrate multiple system design methodologies. Using the System Development Life Cycle (SDLC) or Rapid Application Development (RAD) can build a strong foundation, while Agile methodologies ensure continuous evolution through daily reviews and iterative improvements.

Source: Information System for Business and Beyond

2. Content and Information Architecture

Key IA models include user flows, wireframes, and web footers. User flows help define how users navigate pages, wireframes reveal information gaps, and web footers ensure important details are indexed for easy access.

Source: What is Information Architecture?

3. User Experience and Visual Design

The seven key factors influencing user experience include: useful, usable, findable, credible, desirable, accessible, and valuable. Each factor ensures the product meets users' needs, builds trust, and delivers lasting value.

Source: The 7 Factors that Influence User Experience

4. Misinformation and Disinformation

Understanding unconscious biases and verifying information are critical to avoiding misinformation. Key steps include verifying sources, being skeptical, reading beyond headlines, and recognizing biases like confirmation bias.

Sources:

  • Unconscious Biases That Get in the Way of Inclusive Design
  • Misinformation and Disinformation: A Guide to Protecting Yourself
